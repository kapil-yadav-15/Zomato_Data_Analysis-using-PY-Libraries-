# Zomato_Data_Analysis-using-PY-Libraries-

Conducted Exploratory Data Analysis (EDA) on a Zomato restaurant dataset using Python to uncover patterns in restaurant types, customer 
voting behavior, rating distributions, pricing trends, and the impact of
online vs offline ordering on restaurant performance. The project demonstrates end-to-end data analysis from raw data loading to 
actionable business insights using industry-standard Python libraries.

Data loading and inspection using Pandas (read_csv, head, dataframe exploration)
Data cleaning and type conversion using custom Python functions
Statistical aggregation using GroupBy and SUM
Data visualization using Matplotlib (histogram, line plot) and Seaborn (countplot, boxplot)
